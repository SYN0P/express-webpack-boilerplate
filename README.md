# express-webpack-boilerplate
webpack boilerplate using babel, sass with express server

## Usage
### install
```
npm install
```

### webpack bundling
```
npm rnu build
```

### express server start
```
npm start
```

## Denpendencies
### Webpack
- webpack
- webpack-cli
- html-webpack-plugin
### Babel
- @babel/cli
- @babel/core
- @babel/preset-env
- babel-loader
### SASS
- css-loader
- sass
- sass-loader
- style-loader
### Express
- express
- cookie-parser
- debug
- morgan
- dotenv
